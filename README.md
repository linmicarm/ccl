# CCL Bootcamp
 
My journey through Color-Coded Labs, a 19-week coding bootcamp, including classwork, homework, projects, and everything in between.
 
## How It's Made
 
**Tech used:** JavaScript, HTML, CSS 
 
This repository documents my week-by-week progress through the bootcamp curriculum. Each folder represents a different phase of learning, from foundational concepts to applied projects, capturing both the process and the growth along the way.
 
I use this repo to stay organized, track how far I've come, and reflect on concepts as they build on each other.
 
## What's Inside
 
- **Classwork**: code written during live sessions and lectures
- **Homework**: assigned exercises completed outside of class
- **Projects**: larger builds that put multiple concepts together
- **Notes & Resources**: anything worth saving from the learning process

## Lessons Learned
 
- Writing code daily, even imperfect code, builds intuition faster than anything else
- Revisiting earlier work shows how quickly skills compound over a short period of time
- Asking questions and staying curious matters more than already knowing the answer
- The bootcamp pace pushes you to get comfortable being uncomfortable, and that's the point
